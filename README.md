🏠 HostelMart — Student Marketplace

HostelMart is a frontend student marketplace web application where hostel students can buy and sell second-hand items within their campus.
The platform provides a clean, responsive UI with product listings, search, filtering, and interactive features to make student trading simple and convenient.

Built using HTML, CSS, and JavaScript, the project focuses on frontend UI design, responsive layouts, and dynamic client-side interactions.

✨ Features
🛍️ Marketplace

Browse hostel items like books, electronics, furniture, and more

Product grid with pricing and seller details

Interactive product modal view

🔍 Search & Filtering

Search items by name

Filter by category

Sort by newest, lowest price, highest price

📦 Sell Items

List items with details such as:

Price

Condition

Seller info

👤 User Listings

View My Listings

Manage active and saved items

🛒 Cart System

Add items to cart
